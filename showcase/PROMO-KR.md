## 스레드/인스타 (스토리텔링 버전)

바이브코딩으로 앱 만들 때 가장 답답한 거?
UI가 "돌아는 가는데 촌스러운" 거.

디자이너 고용? 비쌈.
직접 수정? 끝이 없음.

그래서 만들었음.
AI한테 "디자인 감각"을 심어주는 오픈소스.

2,200줄의 디자인 규칙:
→ "숫자는 크게, 단위는 작게, 항상 2:1"
→ "순수 검정 절대 금지. 가장 어두운 색은 #2A2A2A"
→ "카드 그림자는 4-8%. 눈에 보이면 너무 강한 거"
→ "드롭다운 금지. 2-4개면 토글, 5개 이상이면 별도 페이지"

이런 규칙 60개가 들어있음.
Claude Code가 읽으면 토스급 UI를 뽑아냄.

컴포넌트 47개, AI 스킬 10개 포함.
복사 한 번이면 끝.

github.com/bitjaru/styleseed

## 트위터/X (한글)

**트윗 1:**
디자이너 없이 토스급 UI 만드는 오픈소스 냄.

AI한테 디자인 룰 2,200줄 먹이면
바이브코딩으로도 프로 수준 UI가 나옴.

47개 컴포넌트 + 10개 Claude Code 스킬.
복사 한 번이면 끝.

github.com/bitjaru/styleseed

**트윗 2:**
왜 필요하냐면

Claude Code한테 "대시보드 만들어" 하면
간격 제각각, 폰트 뒤죽박죽, 색 남발하는 결과물이 나옴.

근데 "숫자는 48px, 단위는 24px, 항상 2:1"
"카드 안에만 콘텐츠 배치" "검정 금지"
이런 규칙을 줬더니?

토스 앱처럼 나옴.

**트윗 3:**
UX 스킬도 있음

/ux-audit → 닐슨 10대 원칙으로 UX 감사
/ux-copy → 버튼 라벨, 에러 메시지 자동 생성
/ux-feedback → 로딩/에러/빈 상태 자동 추가
/ux-flow → 유저 플로우 설계

디자이너+UX 리서처 역할을 AI가 대신.

**트윗 4:**
awesome-design-md (23K 스타)랑 같이 쓰면 시너지.

DESIGN.md = 브랜드 컬러/폰트 (피부)
StyleSeed = 레이아웃/UX/컴포넌트 (뼈대+근육)

둘 다 복사하면 AI가 브랜드 아이덴티티 + 디자인 감각 둘 다 이해함.

MIT 오픈소스. 새 디자인 스타일 PR 환영.
github.com/bitjaru/styleseed
