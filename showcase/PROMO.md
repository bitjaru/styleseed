## Reddit r/webdev Post

**Title:** Open-sourced 2,200 lines of design rules for AI coding — makes Claude Code/Cursor produce professional UI without a designer

**Body:**
Been working on this for a while. The core insight: AI coding tools produce generic UI because they have components but no design OPINIONS.

StyleSeed fixes this by giving AI a complete design brain:
- 60 specific visual rules (color philosophy, typography hierarchy, spacing system, forbidden patterns)
- 47 React components (shadcn/ui-based)
- 10 Claude Code slash commands including UX skills (Nielsen's heuristics audit, microcopy generation, feedback states)

Example rules that make the difference:
```
- Single accent color. Everything else is grayscale.
- Card shadows at 4-8% opacity. If you can see the shadow clearly, it's too strong.
- No dropdown selectors inside cards. 2-4 options = pill toggle. 5+ = separate page.
- Numbers always big, units always small. 2:1 size ratio. No exceptions.
```

First seed is Toss-style (Tailwind CSS v4, React 18, dark mode). Planning Apple HIG, Linear, Stripe seeds next.

MIT: https://github.com/bitjaru/styleseed
